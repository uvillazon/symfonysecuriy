Editar usuario

Administracion ›› Usuarios ››

Inicialmente el boton de edicion se encuentra bloqueado , se debe seleccionar un registro de usuario para desbloquearlo.

Una vez seleccionado el registro a editar se desbloqueara .

Realizado un click en el boton se visualizara ventana emergente con los datos del registro seleccionado:

Los campos que tienen un asterisco * son obligatorios.

Si el usuario tiene un idproveedor se habilitara la selección de si es base trópico o no tal como se muestra en la imagen

Si no se ingresan los datos obligatorios se visualizara:

Una vez introducidos los datos se debe hacer click en se mostrara un mensaje de confirmacion:

Si todo salio bien saldra el mensaje de confirmacion:

Caso contrario saldra un mensaje de error especifico o referenciado para TI(Tecnologias de Informacion).
