Posee las siguientes secciones:

1: Opciones del Adm. Libreta direcciones como ser: Crear Destinatario, Editar Destinatario, Editar Grupo, Crear Grupo y Adm Grupo destinatario.

2: Sección de busqueda, donde se introducen los criterios para realizar busqueda en el contenido del grid.

3: Sección del grid, donde se muestran todos las direcciones existentes en la Base de Datos, con la opciones:

Ordenar de forma ASC el grid de acuerdo a este campo seleccionado.

Ordenar de forma DESC el grid de acuerdo a este campo seleccionado.

Solo se debe hacer click en la columna que se desea ordenar.

4: Paginacion del grid.

5: Grid grupos existentes,  permite visualizar todos los grupos existentes.

6: Grid relacion direcciones con grupos,  las relaciones de direcciones con grupos. Ej :

La direccion de nombre TEST y apellido TEST muestra datos del grupo como q pertenece al grupo TEST2, tipo de mensaje es original.
